## 🚀 Run Locally

Clone the project

```bash
git clone https://github.com/Udysharma/portfolio-new.git
```

Go to the project directory

```bash
cd portfolio-new
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Create production build

```bash
npm run build
```
