# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Udysharma/portfolio-new.git
```

Navigate into the project

```bash
cd portfolio-new
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build production version

```bash
npm run build
```
